# Code of Conduct

Silo adopts the **Contributor Covenant, version 2.1**, verbatim and in full.

Canonical text: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

Localized translations: <https://www.contributor-covenant.org/translations>

## Enforcement

Report concerns to `conduct@chrisjenkins.dev`. Reports are confidential. The maintainers respond within 7 days and follow the enforcement ladder defined in section "Enforcement Guidelines" of the Contributor Covenant 2.1.

## Scope

Covers every project space: GitHub issues and PRs, code review comments, project chat, related events.
